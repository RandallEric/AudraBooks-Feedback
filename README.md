# Audra Books - Feedback & Issues

Welcome! This repository is for tracking bug reports, feature requests, and general feedback for **Audra Books** - the premium Audiobookshelf client for iOS and Android.

## How to Submit

- **Bug Report**: [Create a bug report](../../issues/new?template=bug_report.md)
- **Feature Request**: [Request a feature](../../issues/new?template=feature_request.md)

## Before Submitting

1. **Search existing issues** to avoid duplicates
2. **Be specific** - include steps to reproduce bugs, or clear descriptions for features
3. **One issue per report** - don't combine multiple bugs or requests
